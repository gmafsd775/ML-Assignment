## Dataset Information

* Dataset Name: Concrete_Data_Yeh
* Total Rows: 1030
* Total Columns: 9

### Input Features (Independent Variables)

1. Cement
2. Slag
3. Fly Ash
4. Water
5. Superplasticizer
6. Coarse Aggregate
7. Fine Aggregate
8. Age

### Target Variable (Dependent Variable)

* csMPa (Concrete Compressive Strength)

## Project Objective

The objective of this project is to analyze the relationship between concrete ingredients and concrete compressive strength. A Machine Learning model is used to predict the compressive strength of concrete based on its composition and age.

## Tools and Technologies

## Project Workflow

1. Load the dataset.
2. Explore and understand the data.
3. Separate input and target variables.
4. Split the dataset into training and testing sets.
5. Train a Machine Learning model.
6. Evaluate model performance.
7. Predict concrete compressive strength.

## Expected Outcome

The trained model predicts the compressive strength (csMPa) of concrete using the provided material composition and age information.
